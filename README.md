# 대구대학교 교통·주차 내비게이션

대구대학교 경산캠퍼스의 차량 흐름, 주차 포화도, 위험지점을 한 화면에서 확인할 수 있는 정적 웹 지도 서비스.

---

## 미리보기

| 구성 요소 | 설명 |
|----------|------|
| 좌측 사이드바 | 레이어 제어 · 주차 포화도 · 출입문 상태 · 전문가 인사이트 · 범례 |
| 지도 (전체 화면) | 주차장 폴리곤 · 교통 방향 라인 · 위험지점 마커 · 게이트 라벨 |
| 우하단 패널 | 캠퍼스 전체 주차 통계 (총 면수 / 사용 중 / 포화율) |
| 상단 배너 | 혼잡시간(08:30–09:30 / 17:30–18:30) 자동 감지 → 권장 경로 표시 |

## 사용 방법

### 사이드바

| 조작 | 방법 |
|------|------|
| 열기 / 닫기 | 좌측 상단 ≡ 버튼 클릭 |
| 레이어 켜기/끄기 | 사이드바 상단 토글 스위치 클릭 |

### 지도 레이어

#### 주차장 현황 (초록 토글)

주차장 구역이 색상 폴리곤으로 표시됨. 클릭 시 상세 팝업.

| 색상 | 의미 |
|------|------|
| 🟢 초록 | 여유 (포화율 < 50%) |
| 🟡 주황 | 보통 (50–79%) |
| 🔴 빨강 | 혼잡 (80%+) |

#### 교통 흐름 방향 (파랑 토글)

도로 세그먼트와 방향 화살표(▲)가 표시됨. 클릭 시 상세 팝업.

| 색상 | 의미 |
|------|------|
| 🔵 파랑 | 일반 구간 |
| 🟡 주황 | 주의 구간 (risk ≥ 1.5) |
| 🔴 빨강 | 고위험 구간 (risk ≥ 2.0) |
| 🟢 초록 | 권장 우회로 (risk < 0.9) |
| 파란 점선 | 일방통행 구간 |

#### 전문가 인사이트 (노랑 토글)

! 원형 마커가 위험지점을 표시함. 클릭 시 전문가 권고 팝업.

| 마커 색상 | 위험 등급 |
|----------|----------|
| 🔴 빨강 | critical — 즉시 개선 필요 |
| 🟠 주황 | high — 단기 조치 필요 |
| 🟡 노랑 | moderate — 모니터링 필요 |
| 🟢 초록 | low — 대안 경로 정보 |

### 혼잡시간 배너

오전 8:30–9:30 또는 오후 5:30–6:30에 접속하면 상단에 권장 경로 배너가 자동으로 나타남.

```
● 혼잡시간 권장 경로
서문 진입 → 외곽 순환로 경유 → 사범대·공과대 방면
중앙 도로 이용 시 +12분 예상
```

| 위치 | 위험 등급 | 권고 |
|------|----------|------|

| 서문 경사로 | moderate | 동절기 결빙 — 열선·염화칼슘 살포 장치 검토 |

## 알려진 제약

- **정적 데이터:** 주차 점유율과 교통 상태는 스냅샷 값으로, 실시간 연동되지 않음
- **좌표 정밀도:** GeoJSON 좌표는 개략적 추정값 — 실 운영 시 GIS 측량 데이터로 교정 필요
- **오프라인 미지원:** CDN(Leaflet, Alpine.js, 폰트, 지도 타일) 인터넷 연결 필요
- **모바일 미최적화:** 데스크탑 기준 레이아웃
